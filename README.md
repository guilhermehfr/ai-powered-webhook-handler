## ✨ Features

- **Stripe webhook simulation** – Pre-loaded mock webhook events generated with Faker for realistic payloads.
- **AI-powered handler generation** – Select a webhook event and get a typed TypeScript handler generated by an LLM based on its payload.
- **Syntax highlighting** – Generated code displayed with highlighting powered by Shiki.
- **Interactive API docs** – Swagger + Scalar available directly from the backend.
- **Type-safe API** – End-to-end type safety with TypeScript, Zod, and Drizzle ORM.
- **Modern UI** – Responsive React interface with Tailwind CSS v4, Radix UI, and TanStack Router.
- **Monorepo** – Unified dev experience with pnpm workspaces.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [pnpm](https://pnpm.io/)
- A [Neon](https://neon.tech/) database
- A [Groq](https://console.groq.com/) API key

### Installation
```sh
git clone https://github.com/guilhermehfr/AI-Powered-Webhook-Handler-Generator.git
cd AI-Powered-Webhook-Handler-Generator
pnpm install
```

### Environment Variables
```sh
cp api/.env.example api/.env
```

Fill in your `api/.env`:
```env
DATABASE_URL=postgresql://<user>:<password>@<host>.neon.tech/<dbname>?sslmode=require
GROQ_API_KEY=your_groq_api_key
```

### Database
```sh
# Run migrations
pnpm --filter api db:migrate

# (Optional) Seed
pnpm --filter api db:seed
```

### Development
```sh
# API
pnpm --filter api dev

# Frontend
pnpm --filter web dev
```

Or run both in parallel:
```sh
pnpm --parallel --filter api dev --filter web dev
```

### Build
```sh
pnpm --filter api build
pnpm --filter web build
```
